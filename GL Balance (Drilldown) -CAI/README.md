# GL Balance (Drilldown) -CAI – Oracle EBS SQL Report

**Oracle E-Business Suite** SQL report from the [Enginatics Library](https://www.enginatics.com/reports/gl-balance-drilldown-cai/) powered by [Blitz Report™](https://www.enginatics.com/blitz-report/).

## Overview
** This report is used by the GL Financial Statement and Drilldown report to show Balance details by Code Combination. **

Shows opening balance, period Dr/Cr/Net, and closing balance for each code combination. Supports drill to journals for further analysis.

## Report Parameters
Ledger ID, Period Name, Balance Type, Translated Flag, Currency Code, Actual Flag, Budget Name, Encumbrance Type, Journal Source, Journal Category, Multiplier
