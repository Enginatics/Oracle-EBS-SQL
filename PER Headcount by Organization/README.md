# PER Headcount by Organization – Oracle EBS SQL Report

**Oracle E-Business Suite** SQL report from the [Enginatics Library](https://www.enginatics.com/reports/per-headcount-by-organization/) powered by [Blitz Report™](https://www.enginatics.com/blitz-report/).

## Overview
Active employee and contingent worker headcount summarized by business group, organization, location, job, grade, and employment category. Includes gender breakdown and average service years and age. Use Excel pivot tables for flexible grouping and analysis.

## Report Parameters
Business Group, Organization, Location, Person Type, Effective Date
