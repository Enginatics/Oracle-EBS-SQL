# PER New Hires – Oracle EBS SQL Report

**Oracle E-Business Suite** SQL report from the [Enginatics Library](https://www.enginatics.com/reports/per-new-hires/) powered by [Blitz Report™](https://www.enginatics.com/blitz-report/).

## Overview
Employees hired within a specified date range, showing hire date, organization, job, position, grade, location, supervisor, employment category, and probation details. Useful for onboarding tracking and new hire analysis.

## Report Parameters
Business Group, Organization, Hire Date From, Hire Date To, Location
