# ONT Order Upload – Oracle EBS SQL Report

**Oracle E-Business Suite** SQL report from the [Enginatics Library](https://www.enginatics.com/reports/ont-order-upload/) powered by [Blitz Report™](https://www.enginatics.com/blitz-report/).

## Overview
Upload to Create, Update, Book and Cancel Sales Orders in Oracle Order Management.
Supports creating new orders, updating existing orders, adding new lines to existing orders, booking orders and cancelling orders.

## Report Parameters
Upload Mode, Operating Unit, Order Number, Customer Name, Account Number, Order Type, Line Type, Open only, Exclude Cancelled, Order Status, Order Action, Line Status, Shippable Flag, Creation Date From, Creation Date To, Line Creation Date From, Line Creation Date To, Order Category, Line Category, Item, Item Type, Schedule Ship Date From, Schedule Ship Date To, Ship From Warehouse, Request Date From, Request Date To, Cancelled Date From, Cancelled Date To, Debug
