# PER Terminations and Separations – Oracle EBS SQL Report

**Oracle E-Business Suite** SQL report from the [Enginatics Library](https://www.enginatics.com/reports/per-terminations-and-separations/) powered by [Blitz Report™](https://www.enginatics.com/blitz-report/).

## Overview
Terminated employees within a specified date range, showing termination date, leaving reason, length of service, last organization, job, position, grade, and supervisor. Useful for turnover analysis and exit tracking.

## Report Parameters
Business Group, Organization, Termination Date From, Termination Date To, Leaving Reason
