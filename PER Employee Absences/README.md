# PER Employee Absences – Oracle EBS SQL Report

**Oracle E-Business Suite** SQL report from the [Enginatics Library](https://www.enginatics.com/reports/per-employee-absences/) powered by [Blitz Report™](https://www.enginatics.com/blitz-report/).

## Overview
Employee absence and attendance records showing absence type, category, dates, duration in days and hours, reason, authorizing person, replacement person, and approval status. Useful for leave management, absence tracking, and compliance reporting.

## Report Parameters
Business Group, Person Full Name, Absence Type, Absence Date From, Absence Date To, Organization
